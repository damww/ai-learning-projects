# MNIST项目安装和运行指南

## 环境要求

- Python 3.8+
- pip 包管理器

## 安装依赖

```bash
# 方法1：使用pip（推荐有sudo权限的用户）
pip install torch torchvision numpy matplotlib

# 方法2：使用pip3
pip3 install torch torchvision numpy matplotlib

# 方法3：使用conda（推荐）
conda install pytorch torchvision -c pytorch
pip install numpy matplotlib
```

## 运行训练

```bash
# 进入项目目录
cd 01-mnist-classification

# 运行训练脚本
python3 train.py
```

## 注意事项

- 第一次运行会下载MNIST数据集（约11MB），需要网络连接
- 训练完成后会生成 `mnist_model.pt` 文件
- 在CPU上训练大约需要2-5分钟
