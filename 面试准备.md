## 💻 技术问题速答

### Python基础

**Q: Python和Java的区别？**
> A: Python是解释型语言，开发快、代码简洁；Java是编译型语言，性能更好。AI领域主要用Python。

**Q: 列表和字典的区别？**
> A: 列表是有序的，用索引访问；字典是无序的键值对，用key访问。查找时字典更快。

**Q: 什么是装饰器？**
> A: 装饰器是Python的一个函数，用来在不修改原函数的情况下扩展它的功能，比如@staticmethod。

---

### AI/机器学习核心

**Q: 什么是机器学习？**
> A: 机器学习就是让计算机从数据中学习规律，而不是人工写规则。比如给猫狗图片，让模型自己学会区分。

**Q: 什么是深度学习？**
> A: 深度学习是机器学习的一种，用多层神经网络（深层）来学习复杂特征。比如图像识别、自然语言处理都用深度学习。

**Q: 什么是训练集、验证集、测试集？**
> A:
> - 训练集：用来训练模型的
> - 验证集：训练时调参用的，防止过拟合
> - 测试集：最后评估模型性能的，模拟真实场景

**Q: 什么是过拟合？怎么解决？**
> A: 过拟合就是模型在训练数据上表现很好，但在新数据上表现差。解决方法：
> - 增加数据量
> - 使用正则化（L1/L2）
> - Dropout
> - 早停

**Q: 什么是损失函数？**
> A: 损失函数用来衡量模型预测和真实结果的差距。训练的目标就是让损失函数最小化。

**Q: 什么是梯度下降？**
> A: 梯度下降是一种优化算法，通过沿着梯度反方向一步步更新参数，最终找到损失函数的最小值。

---

### 深度学习框架

**Q: 你用过哪些AI框架？**
> A: 我主要学习过PyTorch，了解TensorFlow。PyTorch比较灵活，调试调试方便，研究和学习用得多。

**Q: PyTorch中的Tensor是什么？**
> A: Tensor是PyTorch的核心数据结构，类似于NumPy的数组，但可以在GPU上运行，支持自动求导。

**Q: 什么是神经网络？**
> A: 神经网络是模拟人脑神经元结构的一种模型，由输入层、隐藏层、输出层组成，通过层与层之间的权重传递信息。

**Q: CNN和RNN的区别？**
> A: CNN（卷积神经网络）适合处理图像等空间数据，通过卷积操作提取局部特征；RNN（循环神经网络）适合处理序列数据，如文本、语音，能记住前面的信息。

**Q: 什么是Transformer？**
> A: Transformer是一种基于注意力机制的架构，完全依赖注意力机制，不用循环结构，是BERT、GPT等模型的基础。

### 数据处理

**Q: 数据预处理包括哪些步骤？**
> A:
> - 数据清洗（处理缺失值、异常值）
> - 特征工程（提取、转换特征）
> - 归一化/标准化
> - 数据切分（训练集、验证集、测试集）

**Q: Pandas常用操作有哪些？**
> A:
> - `read_csv()` 读取数据
> - `head()` 查看前几行
> - `describe()` 统计信息
> - `isnull().sum()` 检查缺失值
> - `groupby()` 分组统计

**Q: 如何处理缺失值？**
> A:
> - 删除：`df.dropna()`
> - 填充：`df.fillna(value)`
> - 插值：用统计值填充（均值、中位数）
> - 预测：用模型预测缺失值

### 工程能力

**Q: Git常用命令？**
> A:
> - `git clone` 克隆代码
> - `git add` 添加文件
> - `git commit` 提交
> - `git push` 推送到远程
> - `git pull` 拉取更新

**Q: 了解Docker吗？**
> A: 了解基本概念。Docker是容器化技术，可以把应用和依赖打包在一起，保证"一次构建，到处运行"。在AI部署中常用。

**Q: 什么是MLOps？**
> A: MLOps是机器学习运维，涉及模型版本管理、实验跟踪、自动化部署、监控等，确保模型能稳定、可靠地部署到生产环境。

**Q: 理解CI/CD吗？**
> A: 理解。CI是持续集成，自动测试代码；CD是持续部署，自动发布。我知道在AI项目中，可以用CI/CD自动化模型训练和部署流程。

---

### 模型优化

**Q: 什么是模型量化？**
> A: 量化是将模型参数从高精度（如32位浮点）转换为低精度（如8位整数），减少模型体积和计算量，通常精度损失很小。

**Q: 什么是模型剪枝？**
> A: 剪枝是删除模型中不重要的权重或神经元，减少模型参数量，提升推理速度。

**Q: 什么是知识蒸馏？**
> A: 知识蒸馏是用一个大的"教师模型"训练一个小的"学生模型"，让小模型学习大模型的知识，实现轻量化部署。

## 🔑 必背概念速记

| 概念 | 一句话解释 |
|------|-----------|
| 监督学习 | 有标签数据，从输入输出对中学习（分类、回归） |
| 无监督学习 | 无标签数据，找数据中的规律（聚类、降维） |
| 回归 | 预测连续值（比如房价） |
| 分类 | 预测离散类别（比如猫狗识别） |
| Epoch | 完整遍历一次训练集 |
| Batch | 一次训练用的样本数量 |
| Learning Rate | 学习率，控制参数更新的步长 |
| 正则化 | 防止过拟合的技术（L1/L2） |
| Dropout | 随机丢弃神经元，防止过拟合 |
| 激活函数 | 引入非线性，让神经网络能学复杂模式（ReLU、Sigmoid） |
| CNN | 卷积神经网络，适合处理图像等空间数据 |
| RNN | 循环神经网络，适合处理序列数据 |
| Transformer | 基于注意力机制的架构，是BERT/GPT等模型的基础 |
